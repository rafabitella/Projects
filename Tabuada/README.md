# Tabuada

Multiplication Table Project
Description

The Multiplication Table Project is an interactive website designed to help users learn and practice basic math operations, with a focus on the multiplication table.

The application allows users to choose a number and instantly view its multiplication table in a clear and dynamic way, making learning easier and more practical.

Technologies Used
HTML
CSS
JavaScript

How to Use
Open the website in your browser
Choose a number
View the multiplication table generated automatically

Projeto Tabuada
Descrição

O Projeto Tabuada é um site interativo desenvolvido para ajudar usuários a aprender e praticar operações matemáticas básicas, com foco principal na tabuada.

A aplicação permite que o usuário escolha um número e visualize sua tabuada de forma rápida, clara e dinâmica, facilitando o aprendizado de maneira prática.

Tecnologias utilizadas
HTML
CSS
JavaScript

Como usar
Abra o site no navegador
Escolha um número
Visualize a tabuada gerada automaticamente
